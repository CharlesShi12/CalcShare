Calc Share



What is Calc Share?

Calc Share is a calculator that allows for real-time collaboration between users. Essentially, multiple people can use one calculator at the same time with real-time changes.



How do I join a new room?

If you look at the URL of your webpage, there should be # with a combination of letters numbers. That is the ID for your room so for others to join, they need to remove their current code and paste your room's code to the end of the URL.



Syntax
Addition, Subtraction, Multiplication, Division

To add, use + (ex. 2+2). To subtract, use - (ex. 2-2). To multiply, use * (ex. 2*2). To divide, use / (ex. 2/2).

Square and Square Root

To square, use ^ (ex. 2^2). To square root, use sqrt() (ex. sqrt(2)).

Logarithms

To implement natural logarithm, use ln (ex. ln(2)). To implement logarithm, use log (ex. log10(20) is equivalent to log base 10 (20)).

Trigonometry

To use any trigonometry, write the function's abbreviations and specify whether you are using degrees or radians (ex. sec(10 radians)).
